# Remainder Operator  (JavaScript)
Divides the value of a numeric expression by the value of another numeric expression, and produces the remainder.  
  
## Syntax  
  
```  
result = expression1 % expression2  
```  
  
## Arguments  
 `result`  
 Any variable.  
  
 `expression1`  
 Any numeric expression.  
  
 `expression2`  
 Any numeric expression.
