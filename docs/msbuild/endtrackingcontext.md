# EndTrackingContext
End the current tracking context.  
  
## Syntax  
  
```  
HRESULT WINAPI EndTrackingContext();  
```  
  
## Return Value  
 An **HRESULT** with the **SUCCEEDED** bit set if the tracking context was ended.  
  
## Requirements  
 **Header:** FileTracker.h
